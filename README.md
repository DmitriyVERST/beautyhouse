# TWINS BEAUTY HOUSE — Маникюрный салон

# Live: https://dmitriyverst.github.io/beautyhouse/

---

## Описание проекта

Этот проект представляет собой сайт для маникюрного салона **TWINS BEAUTY HOUSE**, выполненный с использованием современных технологий и библиотек для создания стильного и удобного интерфейса. Основная задача — дать посетителям возможность легко ознакомиться с услугами, ценами, мастерами и акциями салона, а также записаться на процедуры онлайн.

На сайте реализованы такие функции, как интерактивные галереи работ, плавные переходы, анимации текста и элементов, а также фильтрация услуг по категориям и ценам. Для отображения галерей и слайдеров используется библиотека **Splide**, что позволяет создавать красивые и отзывчивые слайд-шоу.

---

## Используемые технологии

- **HTML**, **CSS**, **JavaScript** — основы веб-разработки  
- **Splide** — слайдер для отображения галерей, отзывов и услуг  
- **Swiper** — слайдер для других элементов (если есть)  
- **Lenis** — библиотека для плавной прокрутки страниц  
- **Splitting** — анимация текста и элементов  
- **GSAP** — создание анимаций и эффектов переходов  
- **MixItUp** — фильтрация и сортировка услуг и портфолио  
- **jQuery** — работа с DOM и обработка событий
